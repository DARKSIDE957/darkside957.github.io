# VR Help Center

Welcome to the **VR Help Center**, a simple website designed to assist users in setting up and troubleshooting their **Pico VR** and **Quest VR** devices. Whether you're a beginner or need help with advanced configurations, this guide has you covered.

## About

This website provides step-by-step guides for connecting **Pico VR** and **Quest VR** devices. It is built using the **Jekyll Minimal Theme** and hosted on **GitHub Pages**.

The goal of this project is to make VR setup easier for everyone, whether you're new to VR or an experienced user looking for quick troubleshooting tips.

---

## Features

- **Pico VR Setup Guide**: Detailed instructions for setting up Pico VR devices.
- **Quest VR Setup Guide**: Step-by-step guide for Quest VR devices.
- **Minimal Design**: Clean and user-friendly interface powered by the Jekyll Minimal Theme.
